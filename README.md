# ⚽ Trivio AI Services

**Graduation Project** 
Trivio is a football social media platform powered by modern AI technologies including LLMs, Retrieval-Augmented Generation (RAG), Computer Vision, Recommendation Systems, and Natural Language Processing (NLP).

This repository contains the AI backend services that power Trivio's intelligent features.

## Features

- 🤖 Football AI Chatbot
- 🌐 Live web search for football news
- ⚽ Live match scores
- 📅 Team schedules and fixtures
- 📚 RAG-based document retrieval
- 🔍 Semantic search with Qdrant
- 👤 Football player face recognition
- 😊 Arabic content moderation
- 🎯 Personalized recommendation system
- 🧠 Embedding generation pipeline

## Services

### AI Agent
- LangGraph & LangChain
- RAG with Qdrant
- Live web search
- Live scores retrieval
- Team schedules and fixtures
- Conversation memory

### Face Recognition
- InsightFace
- Face embedding generation
- Football player identification

### Recommendation System
- Content-based recommendations
- Semantic embeddings
- Personalized feed generation

### Toxicity Filter
- Fine-Tuned MARBERT
- Arabic offensive content detection

### Embedding Generator
- Generates vector embeddings
- Uploads embeddings to Qdrant
